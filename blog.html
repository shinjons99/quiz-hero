1.Question: Difference between map, filter, find and forEach?

Answer: map and filter are used to transform or filter an array, find is used to find a single element in an array that meets a condition, and forEach is used to execute a function on each element of an array.


2.Question:Difference between localStorage, sessionStorage and cookies?

Answer: localStorage and sessionStorage are used to store larger amounts of data on the client-side, while cookies are typically used to store smaller pieces of data that need to be sent to the server on each request. localStorage and sessionStorage have no expiration date and their data is only accessible on the same browser, while cookies have an expiration date and their data can be accessed by the server.


3.Question: How Event loop works?

Answer:The event loop is a mechanism that allows JavaScript to handle asynchronous tasks in a non-blocking way. It constantly checks for new events in the event queue, processes them, and executes callbacks and promises in the task queue.


Question: Write about quizTimer function?

Question: Write about chooseQuiz function?

Answer: chooseQuiz function updates the user's given answer for a quiz and displays all answers, keeping track of the quiz IDs.
